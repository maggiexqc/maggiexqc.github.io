Astronote

Written in: Swift

Info:

Astronote is my current project, it is set to be the first atomic-style handwritten note-taking app across all iOS and macOS devices. This style of notes allows users to break down information into its core concepts and link them to information already learned in previous notes for better memory recall.

This app includes intuitive gestures and navigation which allows for an easier and faster note-taking experience.

Milestones:

Currently in development, updates and features coming soon!
