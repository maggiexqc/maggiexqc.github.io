Tumor Classifier

Written in: Javascript, Python

Info:

This AI web application was created to identify 3 different types of tumors: Meningioma, Glioma, and Pituitary. The application uses a convolutional neural network (CNN) that classifies MRI brain scans. This CNN has been trained to identify the presence and type of tumor with a 99.8% test accuracy.

This project was created as a part of a group project for ECS 170: Artificial Intelligence at UC Davis. I contributed in early preprocessing, front-end development, and the final presentation.

How it works:

Enter in a MRI brain scan of a .mat file type into our app and the CNN will determine whether there is a tumor present, as well as the type of tumor between Meningioma, Glioma, and Pituitary tumors.
